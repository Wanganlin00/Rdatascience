5.9 AsIs I()

I()函数来自于base包，它是as.is函数的别名。I()函数用于改变对象的类别，指示R在进行操作时应该将该对象视为其原始形式，不对它进行任何转换。

Show the code
lapply(summary(mtcars$mpg), I) 
#> $Min.
#> [1] 10.4
#> 
#> $`1st Qu.`
#> [1] 15.425
#> 
#> $Median
#> [1] 19.2
#> 
#> $Mean
#> [1] 20.09062
#> 
#> $`3rd Qu.`
#> [1] 22.8
#> 
#> $Max.
#> [1] 33.9
lapply(summary(mtcars$mpg), I) |> as_tibble()
Min. 1st Qu. Median Mean 3rd Qu. Max.
